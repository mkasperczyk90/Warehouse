A microservices warehouse-management system: .NET 10 / Aspire backend, a Vite admin SPA, and an Expo
operator terminal. See [`CONTRIBUTING.md`](CONTRIBUTING.md) for how to build, test, and run it, plus
the CI/CD pipeline and the one-time repository settings it needs.
